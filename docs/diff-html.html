Graph Diff HTML Exporter

Visually compare two graph snapshots in an interactive HTML page. Nodes and edges are color-coded by diff status: green = added, red = removed, gray = common.

Features

  • Force-directed D3.js layout with drag, zoom, and pan
  • Color-coded nodes and edges by diff status
  • Filter toggles to show/hide added, removed, or common elements
  • Statistics panel with Jaccard similarity, edit distance, degree changes
  • Hover tooltips showing node ID, status, degree in each graph
  • Dark/light theme toggle
  • Self-contained HTML — no server required

GUI Usage

Click the "Diff HTML" button in the toolbar. Select a second graph file to compare against the currently loaded graph. Choose an output location and the diff visualization will be exported as an HTML file.

Programmatic Usage

GraphDiffHtmlExporter exporter = new GraphDiffHtmlExporter(graphA, graphB);
exporter.setTitle("Network Evolution: Jan → Feb");
exporter.setLabelA("January");
exporter.setLabelB("February");
exporter.setDarkMode(true);
exporter.export(new File("diff.html"));

// Or get the HTML as a string
String html = exporter.exportToString();

How It Works

Uses GraphDiffAnalyzer to compute the structural diff between two graphs, then renders the merged result as an interactive D3.js force-directed graph. Removed edges are shown with dashed lines. The statistics panel shows Jaccard similarity scores, edit distance, and per-node degree changes.
