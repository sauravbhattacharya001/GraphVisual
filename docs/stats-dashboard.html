📊 Graph Statistics Dashboard

Export a self-contained interactive HTML dashboard that visualises multiple graph analytics on a single page using Chart.js.

Features

Key Metrics Cards — vertex count, edge count, density, average degree, average clustering coefficient, connected components
Degree Distribution — histogram showing how vertex degrees are distributed
Top Vertices by Degree — horizontal bar chart of the most connected nodes
Clustering Coefficient Distribution — histogram of local clustering coefficients across 10 bins
Edge Type Breakdown — doughnut chart showing the proportion of each edge type
Degree vs Clustering Scatter — scatter plot revealing the relationship between connectivity and local clustering

Usage

From the GUI

Click the "Stats Dashboard" button in the toolbar, choose a save location, and open the exported .html file in any modern browser.

From Code

GraphStatsDashboard dashboard = new GraphStatsDashboard(graph);
dashboard.setTitle("My Network Dashboard");
dashboard.export(new File("dashboard.html"));

// Or get as a string:
String html = dashboard.exportToString();

Output

The exported HTML file is completely self-contained (Chart.js loaded from CDN). It features:

  • Responsive grid layout that works on desktop and mobile
  • Clean, modern card-based design for key metrics
  • Interactive Chart.js charts with hover tooltips
  • No server needed — just open the file in a browser

API

MethodDescription
GraphStatsDashboard(graph)Create dashboard for the given JUNG graph
setTitle(String)Set the dashboard page title
export(File)Write dashboard HTML to a file
exportToString()Return dashboard HTML as a string
