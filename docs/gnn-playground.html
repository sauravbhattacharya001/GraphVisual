How It Works

📨

1. Message

Each node collects feature vectors from its neighbors (and optionally itself via self-loops).

📦

2. Aggregate

Neighbor messages are combined using the chosen function: mean, sum, or element-wise max.

⚡

3. Transform

The aggregated vector is multiplied by a learnable weight matrix W (4×4), then passed through an activation function.

🔁

4. Repeat

After k layers, each node's features encode information from its k-hop neighborhood — the key insight behind GNNs.
