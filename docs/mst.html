ℹ️ About

A Minimum Spanning Tree connects all nodes with the minimum total edge weight and no cycles.

  • Prim's — grows tree from a start node, always adding the cheapest crossing edge.
  • Kruskal's — sorts all edges by weight, adds them if they don't form a cycle (Union-Find).
