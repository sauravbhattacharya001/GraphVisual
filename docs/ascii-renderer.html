🔤 ASCII Graph Renderer

Render graphs as ASCII or Unicode art directly in the terminal — no GUI or external tools needed.

Features

Force-directed node placement on a text grid
ASCII and Unicode box-drawing edge styles
Node labels with optional degree annotations
Configurable grid size (width × height)
Node highlighting by name or degree threshold
Adjacency list view (alternative to spatial layout)
Degree histogram as horizontal bar chart
Legend with graph statistics and edge type breakdown
Export to file

Quick Start

GraphAsciiRenderer renderer = new GraphAsciiRenderer(graph);
renderer.setWidth(120);
renderer.setHeight(40);
renderer.setUnicode(true);
renderer.setShowDegree(true);
System.out.println(renderer.render());

Render Modes

MethodDescription
render()Spatial force-directed layout on a character grid
renderAdjacencyList()Sorted adjacency list with tree-style formatting
renderDegreeHistogram()Horizontal bar chart of degree distribution

Configuration

MethodDefaultDescription
setWidth(int)100Grid width in characters
setHeight(int)35Grid height in characters
setUnicode(boolean)falseUse Unicode box-drawing characters
setShowDegree(boolean)falseShow degree count next to labels
setShowWeight(boolean)falseShow edge weights
setShowLegend(boolean)trueShow statistics legend below graph
setLayoutIterations(int)200Force-directed layout iterations
addHighlightNode(String)—Highlight specific nodes
setHighlightDegreeThreshold(int)-1Highlight nodes with degree ≥ threshold
