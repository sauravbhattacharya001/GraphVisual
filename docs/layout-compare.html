📐 Graph Layout Comparer

Export your graph as a self-contained HTML page showing 4 layout algorithms side-by-side in a 2×2 grid. Compare how different spatial arrangements reveal different structural properties of the same network.

Features

  • Synchronized hover — hover a node in one panel to highlight it and its neighbors across all four views
  • Zoom & pan — each panel has independent zoom/pan via mouse wheel and drag
  • Edge type coloring — friend, classmate, familiar stranger, stranger, study group
  • Node sizing — nodes scaled by degree centrality
  • Dark/light theme — toggle with one click
  • Zero dependencies — single HTML file, just open in a browser

Usage (GUI)

Click the "Layout Compare" button in the toolbar, choose a save location, and open the exported HTML file in any modern browser.

Usage (Code)

GraphLayoutComparer comparer = new GraphLayoutComparer(graph);
comparer.setTitle("Student Network 2011");
comparer.export(new File("layout-comparison.html"));

When to use which layout?

LayoutBest for revealing
Force-DirectedNatural clusters, community structure, densely connected groups
