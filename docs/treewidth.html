🌲 Treewidth Analyzer

The TreewidthAnalyzer computes upper and lower bounds on the treewidth of a graph and produces a tree decomposition. Treewidth is a fundamental graph parameter that measures how "tree-like" a graph is.

Why Treewidth Matters

  • Algorithm design: Many NP-hard problems (vertex cover, independent set, graph coloring, Hamiltonian path) can be solved in linear time on graphs of bounded treewidth via dynamic programming on tree decompositions.
  • Structure insight: Low treewidth reveals that a graph has a recursive, tree-like structure that can be exploited.
  • Parameterized complexity: Treewidth is the most studied structural parameter in parameterized algorithms.

Computed Metrics

MetricDescription
Upper BoundWidth of the best tree decomposition found via greedy elimination (min-degree and min-fill heuristics).
Lower BoundMaximum of degeneracy (k-core number) and minor-min-width heuristic.
Tree DecompositionSet of bags (vertex subsets) connected by tree edges, satisfying the tree decomposition properties.
Classificationtw=0 → independent set, tw=1 → forest, tw=2 → series-parallel, tw=3 → many problems linear-time.

Usage

// Full analysis with bounds
TreewidthAnalyzer.TreewidthResult result = TreewidthAnalyzer.analyze(graph);
System.out.printf("Treewidth bounds: [%d, %d]%n",
    result.getLowerBound(), result.getUpperBound());
System.out.printf("Exact? %s%n", result.isExact());

// Access the tree decomposition
TreewidthAnalyzer.TreeDecomposition decomp = result.getDecomposition();
for (TreewidthAnalyzer.Bag bag : decomp.getBags()) {
    System.out.println(bag);
}

// Or get a formatted report
System.out.println(TreewidthAnalyzer.report(graph));

Algorithms

Upper Bounds (Greedy Elimination)

Two heuristics are used, and the best result is kept:

  • Min-degree: Always eliminates the vertex with the fewest remaining neighbors. Fast and often effective.
  • Min-fill: Eliminates the vertex whose elimination adds the fewest new edges. Usually produces tighter bounds but is slower.

Each elimination ordering produces a valid tree decomposition. The width is the size of the largest bag minus one.

Lower Bounds

  • Degeneracy: The maximum k-core number. Always ≤ treewidth.
  • Minor-min-width: Iteratively contracts edges of minimum-degree vertices, tracking the maximum minimum degree seen. Often tighter than degeneracy alone.
