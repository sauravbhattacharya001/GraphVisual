💍 Maximum Matching Finder Find maximum matchings with augmenting path visualization

A matching in a graph is a set of edges with no shared vertices. A maximum matching has the largest possible number of edges. This tool visualizes three algorithms: a simple Greedy heuristic, the Augmenting Path (Berge) method for general graphs, and Hopcroft-Karp for bipartite graphs.
