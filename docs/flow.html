How to Use

  • Add nodes: Click canvas in Node mode
  • Add edges: Switch to Edge mode, click source node then target node. Enter capacity.
  • Set source/sink: Use the Source/Sink buttons, then click a node
  • Edit capacity: Double-click an edge label
  • Run: Click Run for full animation, or Step to advance one augmenting path at a time
  • Min-Cut: After completion, cut edges are highlighted in red dashes
