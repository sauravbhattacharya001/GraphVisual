📖 About Random Walks

A random walk on a graph starts at a node and repeatedly moves to a uniformly random neighbor. Key concepts: Cover time is the number of steps to visit every node at least once. On connected, undirected graphs, the stationary distribution converges to π(v) = deg(v) / 2|E| — nodes with more connections are visited more often. The heatmap above shows empirical visit frequency, which should approach the stationary distribution as the number of steps grows.
