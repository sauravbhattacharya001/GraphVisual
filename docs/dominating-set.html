👑 Dominating Set Finder Find minimum dominating sets with greedy & exact algorithms

A dominating set S of a graph G is a subset of vertices such that every vertex is either in S or adjacent to at least one vertex in S. Finding a minimum dominating set is NP-hard, but greedy approximation works well in practice. This tool lets you visualize both approaches step by step.

About Dominating Sets

📡

Wireless Networks

Dominating sets model optimal placement of relay nodes — every device is within one hop of a relay.

🏥

Facility Location

Place facilities (hospitals, fire stations) so every community is adjacent to at least one.

🔒

Network Monitoring

Position sensors to observe all network links — a dominating set monitors every neighbor.

🧬

Biological Networks

Identify essential proteins in PPI networks: a dominating set covers all interaction partners.
