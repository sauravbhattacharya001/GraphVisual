✂️ Graph Partitioning Balanced bisection with Kernighan-Lin optimization

Graph partitioning divides a graph's vertices into two disjoint sets while minimizing the number of edges crossing the partition (the cut size). The Kernighan-Lin (KL) algorithm iteratively swaps vertex pairs to reduce cut size. Click nodes to manually reassign partitions, or let the algorithm find a good bisection.
