🧪 Topology Hypothesis Tester

Statistically test whether a graph matches known topology archetypes — Scale-Free, Small-World, Random, Core-Periphery, or Hierarchical — using null-model comparisons and ensemble scoring.

Overview

The GraphTopologyHypothesisTester answers the question "What kind of network is this?" by comparing observed structural signatures against Monte Carlo-generated null models. Each hypothesis receives a confidence score and p-value estimate, producing a ranked verdict with evidence summaries.

Hypotheses

TopologyModelKey Signatures
Scale-Free Barabási-Albert Power-law degree distribution; few high-degree hubs; low clustering
Small-World Watts-Strogatz High clustering coefficient; short average path length relative to random
Random Erdős-Rényi Poisson degree distribution; low clustering; uniform edge probability
Core-Periphery Borgatti-Everett Dense interconnected core; sparse peripheral nodes attached to core
Hierarchical Ravasz-Barabási Inverse clustering-degree relationship; modular structure at multiple scales

Quick Start

GraphTopologyHypothesisTester tester =
    new GraphTopologyHypothesisTester(graph);

// Run all five hypothesis tests
tester.runAllTests();

// Ranked results (highest confidence first)
List<HypothesisResult> results = tester.getRankedResults();
for (HypothesisResult r : results) {
    System.out.printf("%s: confidence=%.1f%%, p=%.4f, verdict=%s%n",
        r.getTopology().getLabel(),
        r.getConfidence() * 100,
        r.getPValue(),
        r.getVerdict());   // STRONG_MATCH, POSSIBLE, UNLIKELY, REJECTED
}

// Best-fit topology
TopologyType bestFit = tester.getBestFitTopology();
System.out.println("Best fit: " + bestFit.getLabel());

// Confidence for a specific hypothesis
double conf = tester.getConfidence(TopologyType.SCALE_FREE);

// Full text report
String report = tester.generateReport();

Methodology

For each hypothesis, the tester follows a rigorous statistical protocol:

  1. Compute structural signatures from the observed graph (degree distribution, clustering coefficient, average path length, core-periphery fitness, clustering-degree correlation).
  2. Generate null-model ensemble — Monte Carlo random graphs matching the hypothesis model with equivalent size/density.
  3. Compare observed metrics against the null distribution using z-scores to measure how well the observed graph fits each model.
  4. Assign confidence scores and estimate p-values from the empirical null distribution.
  5. Produce a ranked verdict classifying each hypothesis as STRONG_MATCH, POSSIBLE, UNLIKELY, or REJECTED.

Result Structure

HypothesisResult result = results.get(0);

result.getTopology();      // TopologyType enum
result.getConfidence();    // 0.0 - 1.0
result.getPValue();        // statistical p-value
result.getVerdict();       // STRONG_MATCH | POSSIBLE | UNLIKELY | REJECTED
result.getEvidence();      // List<String> of supporting/contradicting evidence
result.getZScores();       // Map<String, Double> per-metric z-scores

Use Cases

  • Network characterization — classify unknown networks in research or forensic analysis.
  • Model validation — verify that a synthetic network generator produces the intended topology.
  • Anomaly detection — a network that doesn't match any known topology may indicate unusual structure worth investigating.
  • Comparative studies — track how a network's topology classification changes over time or across conditions.
