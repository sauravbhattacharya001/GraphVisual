🔬 Graph Motif Finder

The GraphMotifFinder detects and counts common subgraph motifs — the fundamental building blocks that reveal local structural patterns in a network.

Why Motifs Matter

  • Network characterization: Motif frequency profiles act as a fingerprint that distinguishes network types (social, biological, technological).
  • Role discovery: Nodes participating in many triangles are tightly embedded; hub-star nodes are broadcasters or authorities.
  • Clustering insight: The ratio of closed to open triplets gives the global clustering coefficient directly from motif counts.
  • Anomaly detection: Unusual motif distributions can indicate fraud rings, bot networks, or structural irregularities.

Detected Motifs

MotifSymbolDescription
Triangle (3-clique)△ Three mutually connected nodes — tight-knit group
Star (k ≥ 3)★ Hub connected to k leaves — centralized pattern
Open Path (P3)─○─ Middle node bridges two unconnected endpoints — brokerage
Square (C4)□ Four-node cycle — redundancy / alternative routing

Additional Metrics

  • Global clustering coefficient: 3 × triangles / (3 × triangles + open paths)
  • Local clustering coefficient: Per-node ratio of neighbor links to possible neighbor links
  • Top triangle participants: Nodes embedded in the most triangles

Usage

GraphMotifFinder finder = new GraphMotifFinder(graph);
finder.analyze();

// Print motif census
System.out.println(finder.getCensus());

// Get clustering coefficient
double cc = finder.getClusteringCoefficient();

// Full report
System.out.println(finder.generateReport());

// Per-node clustering
Map<String, Double> local = finder.getLocalClustering();

// Top 5 triangle participants
List<Map.Entry<String, Integer>> top = finder.getTopTriangleNodes(5);

Report Output

The generateReport() method produces a formatted text report including:

  • Motif census with counts
  • Global clustering coefficient
  • Sample triangles, stars, and squares
  • Top triangle participants

Performance

  • Triangles: O(V² × max-degree) with sorted vertex iteration
  • Stars: O(V) single pass
  • Open Paths: O(Σ deg²) over all vertices
  • Squares: Bounded to 50,000 results for large graphs
