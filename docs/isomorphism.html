🔄 Graph Isomorphism Checker Determine if two graphs are structurally identical

Draw or type two graphs and check whether they are isomorphic — structurally identical up to vertex relabeling. If isomorphic, the tool shows a valid vertex mapping between them.

Presets:
Draw mode: Click canvas to add nodes. Click two nodes sequentially to add an edge. Right-click a node to delete it.
Edge list mode: Enter edges as pairs like 0-1, 1-2, 2-3 (one graph per box).
