🏝️ Maximum Independent Set Finder Find largest independent vertex sets with greedy, exact & complement-clique algorithms

An independent set S of a graph G is a set of vertices where no two are adjacent. The Maximum Independent Set (MIS) is the largest such set — its size is denoted α(G). Finding it is NP-hard, but this tool lets you visualize three different approaches step by step.
