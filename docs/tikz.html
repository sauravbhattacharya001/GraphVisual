TikZ/LaTeX Graph Exporter

Export your graph as a .tex file that compiles directly with pdflatex or lualatex. Perfect for including network visualizations in academic papers, theses, and presentations.

Features

  • Force-directed automatic layout (computed at export time)
  • Edge type → color mapping using LaTeX xcolor
  • Edge weight → line width scaling
  • Node degree → size scaling
  • Optional legend showing edge types
  • Standalone mode (complete document) or fragment mode (for \input{})

Quick Start

  1. Load your graph in GraphVisual
  2. Click Export TikZ in the toolbar
  3. Choose a filename (e.g., network.tex)
  4. Compile: pdflatex network.tex

Programmatic Usage

TikzExporter exporter = new TikzExporter(graph);
exporter.setTitle("Student Network — Spring 2026");
exporter.setStandalone(false);  // fragment for \input{}
exporter.setShowLegend(true);
exporter.setCanvasWidth(14.0);  // cm
exporter.export(new File("network.tex"));

Including in a Paper

Set standalone(false) to get just the tikzpicture environment, then include it in your paper:

\usepackage{tikz}
\usetikzlibrary{arrows.meta,positioning}
% ...
\begin{figure}[htbp]
  \centering
  \input{network.tex}
  \caption{Student interaction network.}
  \label{fig:network}
\end{figure}

Edge Type Colors

  • Friend (f) — green
  • Familiar Stranger (fs) — blue
  • Classmate (c) — orange
  • Stranger (s) — red
  • Study Group (sg) — violet
